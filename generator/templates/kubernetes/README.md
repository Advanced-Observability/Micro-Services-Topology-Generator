# Kubernetes templates

Templates to generate Kubernetes configuration files.

## Files

- [k8s-ioam-collector-pod.yaml](./k8s-ioam-collector-pod.yaml): Template for the pod containing the IOAM collector;
- [k8s-ioam-collector-service.yaml](./k8s-ioam-collector-service.yaml): Template for the service representing the IOAM collector;
- [k8s-jaeger-pod.yaml](./k8s-jaeger-pod.yaml): Template for the pod containing Jaeger;
- [k8s-jaeger-service.yaml](./k8s-jaeger-service.yaml): Template for the Jaeger service;
- [k8s-meshnet-config.yaml](./k8s-meshnet-config.yaml): Template for Meshnet-CNI config for a pod;
- [k8s-meshnet-interface.yaml](./k8s-meshnet-interface.yaml): Template for an interface for Meshnet-CNI;
- [k8s-pod.yaml](./k8s-pod.yaml): Template for a Kubernetes pod;
- [k8s-service.yaml](./k8s-service.yaml): Tmplate for a Kubernetes service.
